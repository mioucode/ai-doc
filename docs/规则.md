- 当遇到`content_block_start`且`type`为`content_block.tool_use`时，页面显示一个卡片，内容为`displayText`
- 当遇到`content_block_delta`且`delta.type`为`text_delta`时，页面显示一个卡片，内容流式展示`text`
- 当遇到`content_block_stop`且存在`payload`时，页面显示一个卡片，如果包含`payload.displayText`则展示`payload.displayText`，如果包含`payload.plan.steps`则增加展示列表，列表项为`payload.plan.steps[].displayTitle`，如果包含`payload.normalizedResult.items`则增加展示列表列表项为`payload.normalizedResult.items[].title`和`payload.normalizedResult.items[].description`
- `data: [DONE]`流式输出结束
